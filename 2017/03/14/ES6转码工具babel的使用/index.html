Babel工具

Babel CLI 是一个可以在命令行编译ES6的工具

  1. 安装方法

  2. 配置

    Babel是通过安装插件(plugin)或者预设(preset)来编译代码的

  • 先创建一个配置文件 .babelrc ,放置在项目根目录下

  • 安装预设,把ES6编译成ES5.

  • –save 可以把安装的包写进pageage.json的dependencies里,生产环境需要依赖的包

  • –save-dev 可以把安装的包写进pageage.json的devDependencies里,开发测试环境需要依赖的包

  • 修改配置文件

  • 配置完毕,新建一个js文件demo.js,写一段代码实验一下

  • 命令窗口执行

  • 执行后代码变为ES5的代码

  • 配置完毕,借助Babel,在现在的项目中就可以完全释放ES6的魔力了
